# Hou Linzhi — Personal Website

Personal website of Linzhi Hou (侯林之), CS student at Tsinghua University.

## Tech Stack

- **Framework**: Next.js 16 + React 19 + TypeScript
- **Styling**: Tailwind CSS v4
- **Animation**: Framer Motion
- **Deployment**: Vercel
- **Domain**: [houlinzhi.com](https://houlinzhi.com)

## Structure

| Route | Description |
|-------|-------------|
| `/` | Personal homepage (profile, journey, skills, projects, volunteer) |
| `/workspace` | 14 online tools across 3 categories |
| `/workspace/diagram` | Mermaid diagram editor |
| `/workspace/json-formatter` | JSON format & validate |
| `/workspace/color-converter` | HEX/RGB/HSL converter |
| `/workspace/hash-generator` | MD5/SHA hash generator |
| `/workspace/base64` | Base64 encode/decode |
| `/workspace/uuid` | UUID v4 generator |
| `/workspace/timestamp` | Unix timestamp converter |
| `/workspace/regex` | Regex tester with highlighting |
| `/workspace/password` | Password generator with controls |
| `/workspace/url-encode` | URL encode/decode |
| `/workspace/text-stats` | Text statistics counter |
| `/workspace/pomodoro` | Pomodoro focus timer |
| `/workspace/qr-generator` | QR code generator |
| `/arknights` | Arknights headhunting probability calculator |
| `/arknights/headhunting` | Full probability tool (4 tabs) |

## Development

```bash
npm install
npm run dev    # http://localhost:3000
npm run build  # production build
```

## Deployment

Push to `main` branch — Vercel auto-deploys.
